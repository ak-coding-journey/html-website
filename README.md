# html-website
A simple static website built entirely with HTML.   This project demonstrates the fundamentals of web development without relying on CSS or JavaScript.   It includes clean markup, semantic structure, and accessible design practices, making it a great starting point for beginners learning HTML.  



Online Examination Form
This project demonstrates a fully functional HTML-based online certification examination form. It is designed with accessibility, usability, and professional structuring in mind, making it a practical resource for beginners and educators learning form design.

Features
Candidate Information Section  
Collects essential details such as full name, roll number, and email with required field validation.

Section A – Multiple Choice Questions  
Includes radio buttons for single-choice questions and checkboxes for multiple selections.

Section B – Short Answer Questions  
Provides text areas with character limits for concise written responses.

Declaration Section  
Ensures candidates confirm honesty before submission.

Form Controls

Submit button to finalize the exam

Reset button to clear all responses



Accessibility & Best Practices
Proper use of <fieldset> and <legend> for logical grouping.

Labels linked to inputs via for attributes.

Required fields and validation for critical inputs.

Semantic HTML elements for clarity and structure.



Usage
Open form_8.html in any modern browser to attempt the exam. The form is lightweight, responsive, and requires no external dependencies.
